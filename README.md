# Run and deploy PantryPal
[https://ai.studio/apps/aa987ba4-0489-4e93-807b-25925dbcec1c](https://pantrypal-252908779850.us-central1.run.app/)

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
